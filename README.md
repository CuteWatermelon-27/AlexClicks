 Alex Jeffery Photography Website

Kaushik Murali

This website was created for a photography client named Alex Jeffery. The site is made to showcase portrait and lifestyle photography and allow users to learn about the photographer and book a session. The website includes a home page, about page, and contact page.

Features Implemented

Responsive Navigation
I added a hamburger menu that appears on smaller screens. When the user clicks it, the navigation links open and close.

Form Validation
The contact form checks if the name, email, and message are filled correctly before submitting. If something is wrong it will show an error.

Back to Top Button
A button appears when the user scrolls down the page. Clicking it smoothly scrolls the user back to the top.

Accordion FAQ
I added a FAQ section where users can click a question to show or hide the answer. This helps reduce too much text on the page.

Fetch API
The website loads a random quote from an online API and displays it on the homepage.

Design Changes From Assignment 1

Some layout and spacing changes were made to improve readability and make the site look cleaner. I also improved the navigation and added more sections like FAQ and quote to make the site more interactive.

AI Usage

I used AI tools like ChatGPT mainly for help with CSS styling and some JavaScript structure. It helped me understand how to organize my HTML and connect my JavaScript file to the pages. I also used it to understand how features like the hamburger menu, form validation, and the back to top button work.

The AI suggestions was helpful when I got stuck with some parts of the code and when things were not working. It helped explain the logic and give examples so I could fix the problems myself. I learn more about how JavaScript interacts with HTML and CSS from using it.
